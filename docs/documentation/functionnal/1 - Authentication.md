# Authentication

SympAuthy is an authentication server compliant with the [OAuth 2](https://datatracker.ietf.org/doc/html/rfc6749) authorization framework.

## End-user authentication

The authentication of end-user is supported through OAuth2 [authentication code grant flow](https://datatracker.ietf.org/doc/html/rfc6749#section-4.1).

### Default authentication flow

The end-user will be prompted to:
- sign up/sign-in with an email and a password. (if password authentication is enable by config)
- use a third party authentication service.

## Service account

Service account allows a non-interactive service to authenticate to this authorization server to access your application.
